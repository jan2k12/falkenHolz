This plugin insert a counter script into your wordpress blog. There are several designs and configuration possibles. You also can run this counter with enabled wordpress caching!

== Description ==

 - supportet admininterfacelanguages: german, english
 - graphic statistic [example](http://www.phpwelt.net/instantcounterstats-9201-0-0-en.html)
 - detecting  search keywords from visitors that came via searchengine
 - prefabricated designs, or your own
 - free adjustable reloadbarrier
 - you can define html-code what it posted bevor und behind your counter -> you can center your couter etc.
 - working also with cached blogs

No Fee Charged. No Account Required. No Strings Attached.

== Installation ==

1. Upload `counter.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. go to your widget interface and add the "counter"-Widget
